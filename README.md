# Sistema de Biblioteca em Java

Este é um projeto simples de sistema de biblioteca desenvolvido em Java, com o objetivo de praticar conceitos de lógica de programação e Programação Orientada a Objetos (POO).

## Sobre o projeto

O sistema funciona via terminal e permite ao usuário gerenciar livros de forma simples, realizando operações como cadastro, busca, empréstimo e devolução.

Este projeto foi desenvolvido como parte do meu aprendizado inicial em Java, com foco na aplicação prática dos conceitos estudados.

---

## Conceitos aplicados

- Lógica de programação
- Programação Orientada a Objetos (POO)
- Criação de classes e objetos
- Uso de arrays de objetos
- Métodos com retorno (`boolean`)
- Estruturas de repetição (`for`, `while`)
- Estruturas condicionais (`if/else`)
- Entrada de dados com `Scanner`

---

## Funcionalidades

- Cadastrar livros
- Listar livros cadastrados
- Buscar livro pelo nome
- Alugar livro
- Devolver livro
- Encerrar o sistema
